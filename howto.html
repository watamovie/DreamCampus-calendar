使い方

  1. iOSショートカットを実行すると、UTF-8 Base64 化した JSON を URL エンコードして本アプリが開きます。
    例: encodeURIComponent(btoa(unescape(encodeURIComponent(json))))
  2. URL から読めない場合はクリップボード読込が試行されます。失敗したらペーストボタンを押すかテキストボックスに貼り付けます。
  3. 欠落している項目を編集してICS生成ボタンを押します。
  4. 編集内容は自動でブラウザに保存され、次回アクセス時に復元されます。
